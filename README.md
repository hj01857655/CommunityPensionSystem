# community-pension-system
    这是社区养老系统
## 项目简介
    基于Spring boot+Vue的社区养老系统的设计与实现
## 项目技术
    SpringBoot3.4.3+Vue+Vue-Router4+Element-Plus+Axios+Echarts+Webpack+Vite
## 项目运行
    1. 克隆项目
    2. 安装依赖
    3. 运行项目
    4. 打开浏览器
    5. 输入地址
    6. 登录
    7. 开始使用
